# 怎么实现一个宽高自适应的正方形？

## 参考答案

- 利用vw来实现：

```css
  width: 10%;
  height: 10vw;
  background: tomato;
}
```

```css
  width: 20%;
  height: 0;
  padding-top: 20%;
  background: orange;
}
```

```css
  width: 30%;
  overflow: hidden;
  background: yellow;
}
.square::after {
  content: '';
  display: block;
  margin-top: 100%;
}
```
