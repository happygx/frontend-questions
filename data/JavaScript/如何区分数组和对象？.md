# 如何区分数组和对象？

## 参考答案

## 方法1 ：通过 ES6 中的 Array.isArray 来识别

```
console.log(Array.isArray({}))//false
```

```
console.log({} instanceof Array)//false
```
```
console.log({}.constructor)//[Function: Object]
```

```
console.log(Object.prototype.toString.call({}))//[object Object]   
```
