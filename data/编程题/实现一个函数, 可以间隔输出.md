# 实现一个函数, 可以间隔输出

## 参考答案

可以使用 JavaScript 中的定时器函数 setInterval 来实现，具体如下： 

```js
	let count = 0;

	return (param) => {
		const timer = setInterval(() => {
		    fn(param)
		    count++;
		    if (count >= repeat) {
		      clearInterval(timer);
		    }
	    }, interval * 1000);
	}
}
```
