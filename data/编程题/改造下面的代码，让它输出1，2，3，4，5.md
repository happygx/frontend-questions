# 改造下面的代码，让它输出1，2，3，4，5

## 参考答案

解决方法：

* 利用IIFE(立即执行函数表达式)当每次for循环时，把此时的i变量传递到定时器中

```js
  (function(j){
    setTimeout(function timer(){
    	console.log(j)
    }, 0)
  })(i)
}

```

```js
  setTimeout(function timer(j){
  	console.log(j)
  }, 0, i)
}
```
```js
  setTimeout(function timer(){
 	 console.log(i)
  },0)
}

```
